## 步骤

> 每个步骤先输出标题，再输出结果

- 逻辑正确：检查逻辑是否符合需求，边界条件是否已处理（空值、极值、异常输入）
- 规范符合：按 `coding-standards` 逐项检查，输出每项检查结果
- 安全检查：检查是否有敏感信息硬编码，用户输入是否有校验
- 文档同步：若修改了公共模块，检查使用文档和演示入口页面是否已同步更新
- 异步状态：检查所有异步操作是否有加载状态
- 代码清洁：检查是否有未使用的变量、函数、import 等死代码
- 无重复：检查是否有重复逻辑和代码，可复用部分是否已提取
- 可读性：检查命名是否清晰，函数和类是否有注释，复杂逻辑是否有过程说明
