# 代码 Review

> 通用规则见 `base.review-base`

## 检查清单
- 逻辑正确：逻辑符合需求，边界条件已处理（空值、极值、异常输入）
- 规范符合：按 `base.coding-standards`的要求逐项检查，输出每项检查结果
- 安全检查：无敏感信息硬编码，用户输入有校验
- 文档同步：修改公共模块时，使用文档和演示入口页面已同步更新
- 异步状态：所有异步操作有加载状态，loading 态不可缺失
- 代码清洁：无未使用的变量、函数、import 等死代码
- 无重复：无重复逻辑和代码，可复用部分已提取
- 可读性：命名清晰，逻辑易于理解；函数和类有注释，复杂逻辑有过程说明
